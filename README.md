# Projeto-Robo

Para ativar digite siga os passos a abaixo:

Garanta que está instalado o Node.js e Python
## Ativar servidor frontend

1º Vá a pasta do arquivo "cd projeto-robo/frontend/projeto-robo"

2º rode no cmd "npm install"

3º rode "npm run dev" para iniciar

## Ativar servidor backend
0º Crie um ambiente virtual e instale as dependencias "requirements.txt"

1º Abra o cmd vá arquivo "cd projeto-robo/back" e rode "Python main.py" no terminal
