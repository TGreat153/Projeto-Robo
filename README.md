# Projeto-Robo

Para ativar digite siga os passos a abaixo:

Garanta que está instalado o Node.js e Python

Vá a pasta do arquivo "projeto-robo/frontend/projeto-robo" e rode no terminal "npm install" de depois "npm run dev"

Crie um ambiente virtual e instale as dependencias "requirements.txt"

Depois em outro terminal vá arquivo "projeto-robo/back" e rode "main.py" no terminal
